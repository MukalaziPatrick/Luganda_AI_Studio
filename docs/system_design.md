# Luganda AI Studio ó System Design

## Stack
- Frontend: React + TailwindCSS
- Backend: FastAPI (Python)
- AI Layer: Local model (NLLB-200) + Claude API fallback
- Retrieval: ChromaDB vector store over Luganda datasets
- Database: SQLite (MVP)

## Machine
- Windows 64-bit, Intel i7-11800H, 16GB RAM, NVIDIA RTX 3050 (4GB VRAM)
